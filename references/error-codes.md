# 错误编码体系全表

## 编码格式

`[大类字母][小类编号][严重度]`
例：`C3!`（致命的逐字直译）、`G4°`（一般的冠词错）、`S4~`（句式优化项）

## 严重度三档

处理优先级铁律：**始终按 `!` → `°` → `~` 顺序解决**。

| 符号 | 含义 | 影响 |
|:---:|---|---|
| `!` | 致命错 | 直接拉低评分档位 |
| `°` | 一般错 | 扣印象分但不致命 |
| `~` | 优化项 | 不算错，但有更地道/更高分写法 |

## 默认严重度判定规则

| 严重度 | 默认归属 | 说明 |
|:---:|:---|:---|
| `!` | G1 主谓一致、G8 句子破碎、严重 C3 逐字直译、L4 生造词 | 直接拉低档位 |
| `°` | G3 单复数、G4 冠词、G6 介词、C1 动名搭配、C2 形名搭配、C5 介词搭配、L1 拼写、L2 词性、M1 大小写、M2 标点 | 扣印象分 |
| `~` | S4 句式单一、C4 轻度冗余、可优化但不算错的表达 | 加分项 |

> **注意：** 同一类别严重度随语境浮动，以「是否拉低评分档位」为最终判据。例如 C3 轻度直译可能只是 °，但导致句子彻底失真的 C3 就是 !。

## G — 语法 Grammar（地基，优先级最高）

| 编码 | 类别 | 典型错误 |
|:---:|------|---------|
| G1 | 主谓一致 | `a number of foreigner **is**...` |
| G2 | 时态 / 语态错误 | 该用完成时却用一般现在时 |
| G3 | 名词单复数 / 可数性 | `many **information**`、`foreigner` 不加 s |
| G4 | 冠词（a/an/the/零冠词） | 该加 `the` 漏掉、可数名词裸用 |
| G5 | 代词指代混乱 | `it` / `they` 指代不清或不一致 |
| G6 | 介词误用 | `depend **of**`、`good **in**` |
| G7 | 非谓语错误 | `By **watch**`（应为 `watching`）、不定式/分词误用 |
| G8 | 句子破碎 | 缺谓语、run-on 流水句、残句 |

## C — 搭配与中式英语 Chinglish（主战场）

| 编码 | 类别 | 典型错误 |
|:---:|------|---------|
| C1 | 动名搭配错误 | `**produce** interest`（应 `develop` / `cultivate`） |
| C2 | 形名搭配错误 | `**thick** interest`（应 `strong` / `keen`） |
| C3 | 逐字直译 | `more and more`、`let the world like China` |
| C4 | 中式冗余堆砌 | 啰嗦、同义反复、套话堆叠 |
| C5 | 介词搭配错 | `interest **to**`（应 `interest in`） |

## L — 词汇 Lexis

| 编码 | 类别 | 典型错误 |
|:---:|------|---------|
| L1 | 拼写错误 | `calligraphy` 拼错 |
| L2 | 词性误用 | `justice and **brave**`（应 `courage`） |
| L3 | 选词不当 | 词义偏差、近义词误选 |
| L4 | 超纲词 / 生造词 | 自创单词、`handwriting` 代 `calligraphy` |

## S — 句式结构 Syntax

| 编码 | 类别 | 典型错误 |
|:---:|------|---------|
| S1 | 特殊句式错误 | 倒装 / 强调 / 虚拟语气用错 |
| S2 | 从句使用错误 | 定语 / 状语 / 名词性从句结构错 |
| S3 | 并列不平行 | `and` 连接的成分结构不一致 |
| S4 | 句式单一 | 通篇简单句，无变化（优化项常见） |
