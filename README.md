# 考研英语写作训练教练 (Kaoyan Writing Coach)

一个融合考研大纲 5500 词和英一写作专业资料的 AI Skill，兼容 **Claude Code**、**OpenClaw** 等平台，为考研英语一考生提供**出题→批改→深挖→微靶→回归**全流程智能教练。

### 各模块说明

| 模块 | 用途 | 调用时机 |
|---|---|---|
| `SKILL.md` | Skill 定义、铁律、检查目录、完整教学逻辑 | AI 平台自动加载 |
| `scripts/history_mgr.py` | 历史数据管理（SQLite 后端），替代手动 JSON 操作 | 每轮训练读写档案、按日期复习错题 |
| `data/sentences.json` | 出题句库（含难度、主题、来源标签） | 出题时由 `query_sentences.py` 检索 |
| `references/error-codes.md` | 系统化错误分类编码（6大类25小类） | 批改时对照编码 |
| `references/vocab-5500.md` | 大纲词汇速查 | 出题/批改前自检词汇超纲 |
| `references/template-*.md` | 范文模板体系（方法论→构建→范文→精讲） | 学生要求讲解模板时 |
| `scripts/query_sentences.py` | 从句库按条件检索句子生成题目 | 每次出题流程强制调用 |
| `scripts/vocab_check.py` | 批量检查词汇是否在大纲范围内 | 出题后 / 批改推荐词时 |

## 核心能力

- **出题 v2.0**：基于考研大纲 5500 词 + 164 条结构化句子库，按六大主题（T1–T6）和五大训练环节（现象描述→原因分析→影响论述→对策建议→价值升华）自动出题
- **批改**：对作文/翻译进行评分和详细逐句批改，6大类25小类错误编码（G语法/C搭配/L词汇/S句式/D篇章/M机械）
- **错误编码反馈**：三档严重度（!致命/°一般/~优化），每次聚焦**最多 5 条致命错误**
- **深挖 → 微靶**：针对每条致命错生成专项微靶练习，逐个击破
- **回归实战**：微靶通关后回归原题型，验证迁移效果
- **学情追踪 v2.1**：Python + SQLite 后端（`scripts/history_mgr.py`），全自动管理 error_stats / wrong_words / zero_streak / graduation
- **每日题目记录**：记录每一天出的全部题目和错误情况（v2.1 新增）
- **按日期复习错题**：支持「复习X月X日的错题」「重做某天错题」按天回顾和重新练习（v2.1 新增）
- **作文模板系统**：内置四大模块（方法论/DIY搭建/模板库/真题范文精析），按需调用

## 适用对象

- 考研英语一备考学生
- 当前作文约 9 分、目标 20 分的考生（默认配置，可调整）
- **不适用于**：四六级、雅思、托福作文备考

## 安装

兼容所有支持 Claude Skill 格式的 AI 平台（Claude Code、OpenClaw 等），将整个目录放入对应平台的 skills 目录即可。

**Claude Code：** 放入 `.claude/skills/kaoyan-writing-coach/`

**OpenClaw / 其他平台：** 按平台的 Skill 加载路径放置，保证 `SKILL.md` 与 `data/`、`references/`、`scripts/` 的目录结构完整：

```
<skills目录>/kaoyan-writing-coach/
├── SKILL.md
├── data/           # profile.json + sentences.json + ...
├── references/     # 词汇表 + 错误码 + 模板库
└── scripts/        # Python 辅助脚本（含 history_mgr.py）
```

### 首次使用（v2.1 新增）

```bash
# 1. 初始化 SQLite 数据库
python scripts/history_mgr.py init

# 2. 迁移已有档案
python scripts/history_mgr.py migrate data/profile.json

# 3. 验证数据完整性
python scripts/history_mgr.py db check
```

## 设计理念

> **参谋型教练** — 分析学情、给出建议，但关键决策（主题、难度、训练类型）始终交给学生拍板。

五条核心铁律：

1. **词汇守界** — 所有参考译文和推荐词汇必须落在考研大纲 5500 词内
2. **参谋不越权** — 主题、难度、训练类型，每次问学生一句
3. **反馈节流** — 致命错上限 5 条，超过折叠，逐个击破
4. **数据驱动** — 每次关键节点调用 `history_mgr.py` 同步数据，不凭感觉
5. **无跨会话记忆** — SQLite 数据库为唯一事实来源，不依赖对话内临时记忆
