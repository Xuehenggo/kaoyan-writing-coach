# 考研英语写作训练教练 (Kaoyan Writing Coach)

一个融合考研大纲 5500 词和英一写作专业资料的 AI Skill，兼容 **Claude Code**、**OpenClaw** 等 Claude 生态平台，为考研英语一考生提供**出题→批改→深挖→微靶→回归**全流程智能教练。

### 各模块说明

| 模块 | 用途 | 调用时机 |
|---|---|---|
| `SKILL.md` | Skill 定义、铁律、检查目录、完整教学逻辑 | AI 平台自动加载 |
| `data/profile.json` | 学情档案：错词列表、弱项标记、成绩历史、训练进度 | 每轮练习必读/写 |
| `data/sentences.json` | 出题句库（含难度、主题、来源标签） | 出题时由 `query_sentences.py` 检索 |
| `references/error-codes.md` | 系统化错误分类编码 | 批改时对照编码 |
| `references/vocab-5500.md` | 大纲词汇速查 | 出题/批改前自检词汇超纲 |
| `references/template-*.md` | 范文模板体系（方法论→构建→范文→精讲） | 学生要求讲解模板时 |
| `scripts/query_sentences.py` | 从句库按条件检索句子生成题目 | 每次出题流程强制调用 |
| `scripts/vocab_check.py` | 批量检查词汇是否在大纲范围内 | 出题后 / 批改推荐词时 |

## 核心能力

- **出题**：基于考研大纲 5500 词 + 真题句库，生成大作文和翻译练习题（由脚本检索，非自由发挥）
- **批改**：对作文/翻译进行评分和详细逐句批改
- **错误编码反馈**：系统化分类错误（语法、词汇、中式英语、句式等），每次聚焦 **最多 5 条致命错误**
- **深挖 → 微靶**：针对每条致命错生成专项微靶练习，逐个击破
- **回归实战**：微靶通关后回归原题型，验证迁移效果
- **学情追踪**：自动记录学习进度、弱项、错词至 `profile.json`，数据驱动教学决策

## 适用对象

- 考研英语一备考学生
- 当前作文约 9 分、目标 20 分的考生（默认配置，可调整）
- **不适用于**：四六级、雅思、托福作文备考

## 安装

兼容所有支持 Claude Skill 格式的 AI 平台（Claude Code、OpenClaw 等），将整个目录放入对应平台的 skills 目录即可。

**Claude Code：** 放入 `.claude/skills/kaoyan-writing-coach/`

**OpenClaw / 其他平台：** 按平台的 Skill 加载路径放置，保证 `SKILL.md` 与 `data/`、`references/`、`scripts/` 的目录结构完整：

```
<skills目录>/kaoyan-writing-coach/
├── SKILL.md
├── data/           # profile.json + sentences.json + ...
├── references/     # 词汇表 + 错误码 + 模板库
└── scripts/        # Python 辅助脚本
```

## 设计理念

> **参谋型教练** — 分析学情、给出建议，但关键决策（主题、难度、训练类型）始终交给学生拍板。

三条核心铁律：
1. **词汇守界** — 所有推荐词汇必须在考研大纲 5500 词内
2. **反馈节流** — 致命错每次上限 5 条，逐个击破不倾倒
3. **数据驱动** — 所有决策基于 `profile.json`，不凭感觉
